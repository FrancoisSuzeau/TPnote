# FrancoisSuzeau/TPnote

##Comment j'ai compilé : 
* J'utilise la commande gradle run --args "chiffre romain ou arabe" pour lancer le programme
* J'utilise la commande gradle test pour lancer tout les tests

###Test fonctionnant :
* Tout les tests fonctionnent hormis le test d'échec pour des valeurs avec des antécédants incorrects, je n'ai pas réussi à la coder.
